## graphql study
## 브렌치별 마이그레이션 확인

```
git clone https://github.com/rg5668/express-graphql.git
```

```
npm install
```

```
node server.js
```

#### http://localhost:4000/graphql 에서 작성된 Schema 확인 가능합니다.
